什么是子路由

这个从名字就能看出它的含义:路由的子级路由.听一位大神说过一句话,子路由又叫菜单路由.

子路由是用来做什么的?

子路由一般用在一个页面有他的基础模版，然后它下面的页面都隶属于这个模版，只是部分改变样式.比如淘宝列表页,点击商品栏目会出现相应的商品内容.
